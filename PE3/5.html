シミュレーション結果の可視化

1. 今日の目標

  • 解析の結果を確認する
  • 解析結果を可視化する
  • 必要があればパラメータを修正し、再解析を行う

2. 解析結果

前回の設定に基づき実施した各都市の解析結果を配布する。
ファイル容量が非常に大きいため、ギガファイル便を使用して配布を行う。自身の担当する都市のボタンを選択し、データをダウンロードすること。

北九州 波佐見町 長岡 広島

ログファイルの確認

ダウンロードしたフォルダ内にある Run.out ファイルをテキストエディタ等で開き、解析が正常に終了しているかを確認する。

Run.out の確認ポイント

  • Calculation Time : 解析にかかった時間が記載されている。
  • Error / Warning : エラー等の記述がないか確認する。
  • Step数 : 設定したステップ数まで計算が進んでいるか確認する。

3. 結果の可視化

解析結果として得られたVTKファイルを使用し、ParaViewを用いて流れの可視化を行う。
ParaViewがインストールされていない場合は、以下のリンクよりダウンロードすること。

ParaView ダウンロード

可視化の手順

基本的な可視化の流れは以下の通りである。

1. ファイル読み込み

ParaViewを起動し、ダウンロードしたVTKフォルダを選択してデータを読み込む。読み込み時は「Apply」ボタンを押すのを忘れないこと。

2. 断面の作成 (Slice)

必要に応じて「Slice」フィルタを使用し、特定の位置での断面を表示する。建物の影響を見るためにはZ軸（高さ方向）での調整が重要となる。

3. ベクトル表示 (Glyph)

風向・風速を矢印で表現するために「Glyph」フィルタを適用する。矢印の密度やサイズを調整し、流れを分かりやすく可視化する。

4. カラーマップ調整

流速などの変数をカラーマップで表示する。値のレンジ（最小値・最大値）を適切に設定し、結果の分布を明瞭にする。

5. 画像・動画保存

可視化した結果を画像（Screenshot）またはアニメーションとして保存する。

4. 修正・提出

解析結果が意図したものと異なる場合（例：流入位置がずれている、解析が途中で止まっている等）は、モデルまたはXMLファイルを修正する必要がある。
修正を行った場合は、再度ファイルを以下のGoogleドライブにアップロードすること。

※特に修正の必要がない場合は、可視化した画像を提出準備として保存しておくこと。
