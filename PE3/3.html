「3Dモデルの作成」

1. 今日の目標

  • 解析対象地を決める。
  • 3Dモデル（stlファイル）を作成する。

2. 3Dモデル作成の手順

オープンデータであるPLATEAUから解析用の3Dモデルを作成する。

プロセス

大まかな流れは以下の通りである。

1. 環境構築

「Unity」をインストールし、PLATEAUデータを扱うためのアセット「PLATEAU SDK for Unity」を導入する。また、モデル変換用に「Blender」もインストールする。

2. データ取得

PLATEAUのサイト（G空間情報センター）から、解析対象としたい都市の3D都市モデルデータをダウンロードする。

3. 範囲選択

UnityでPLATEAUデータを読み込み、SDKの機能を使って解析したい範囲を選択する。目安はグリッド4個分とする。

4. OBJ出力

Unity上で選択・抽出した範囲のモデルを、汎用的な3D形式であるOBJファイルとしてエクスポートする。

5. STL変換

BlenderでOBJファイルを読み込み、地形モデル（DEM）と建築物モデルを分離し、それぞれ別々のSTLファイルとして出力する。

3. 詳細・提出

作業に必要なソフトウェアおよびデータは以下のリンク先から入手すること。

なお、各ソフトウェアのバージョン選定には注意が必要である。特にUnityとSDKの組み合わせについては以下を厳守すること。

推奨環境・バージョン
  • 想定Unityバージョン (動作確認済) : Unity 2022.3.25f1
  • 推奨範囲 : Unity 2022.3.25f1 以上、Unity 6 未満
    ※ Unity 6 には未対応であるため使用しないこと。
  • 対応 SDK バージョン : PLATEAU SDK Toolkits for Unity 2.2.1
